# descriptor-rework — Testing Plan

End-to-end test checklist for the descriptor rework. Runs on either:
- the **desktop simulator** (webcam build, `just sim-webcam <board>`), or
- a **fresh physical device** (ESP32-P4 Waveshare; `just flash <board>`).

Both start with empty flash + SD. The plan walks you through PIN setup,
mnemonic entry, and every descriptor load as explicit steps. Tests run
linearly — state accumulates in the expected way, no resets required
except §10 (which is explicitly destructive).

Legend: `[ ]` pending · `[x]` pass · `[~]` skipped.

Reference values below assume the BIP39 test mnemonic
`abandon abandon abandon abandon abandon abandon abandon abandon
abandon abandon abandon about` (mainnet fingerprint `73c5da0a`).
